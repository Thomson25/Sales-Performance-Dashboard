# Sales-Performance-Dashboard
An interactive business intelligence dashboard developed using Microsoft Power BI to analyze sales performance, profitability, customer segments, and country-wise trends through dynamic visualizations and KPI reporting.

## Overview
This Power BI dashboard analyzes:
- Total Sales
- Total Profit
- Units Sold
- Sales Trends
- Country-wise Sales
- Segment Analysis

## Tools Used
- Power BI
- DAX

## Dashboard Features
- Interactive slicers
- KPI cards
- Monthly profit analysis
- Geographic sales analysis
- Product profitability tracking


## Key Insights
- Segment contributes highest sales.
- Paseo product generates maximum profit.
- Year-end months show peak profit performance.
